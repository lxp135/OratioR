# 贡献者公约行为准则

## 我们的承诺

为了营造一个开放和友好的环境，我们作为贡献者和维护者承诺，让每个人参与本项目及社区都不会遭受骚扰，无论其年龄、体型、残疾状况、种族、性别认同与表达、经验水平、国籍、个人外貌、种族、宗教或性取向如何。

## 我们的标准

有助于营造积极环境的行为示例包括：

* 使用友好且包容的语言
* 尊重不同的观点和经验
* 谦逊地接受建设性批评
* 关注对社区最有利的事情
* 对其他社区成员表现出同理心

参与者不可接受的行为示例包括：

* 使用带有性暗示的语言或图像，以及不受欢迎的性关注或挑逗
* 恶意挑衅、侮辱/贬损性评论，以及人身或政治攻击
* 公开或私下骚扰
* 未经明确许可，发布他人的隐私信息，例如物理地址或电子邮箱
* 其他在专业环境中可被合理视为不当的行为

## 我们的责任

项目维护者有责任阐明可接受行为的标准，并期望对任何不可接受行为采取适当和公正的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、Wiki 编辑、问题和其他贡献，或对贡献者因其认为不当、具有威胁性、冒犯性或有害的其他行为进行临时或永久封禁。

## 适用范围

本行为准则适用于项目空间内，以及个人代表项目或其社区的公共场合。代表项目或社区的示例包括使用官方项目邮箱、通过官方社交媒体账号发布内容，或在线上或线下活动中担任指定代表。项目的代表性可由项目维护者进一步定义和阐明。

## 执行

如遇滥用、骚扰或其他不可接受的行为，可通过联系项目团队（antony@corbetts.org.uk）进行举报。项目团队将审查和调查所有投诉，并根据情况做出其认为适当的回应。项目团队有义务对举报者身份保密。具体执行政策的更多细节可另行发布。

未真诚遵守或执行本行为准则的项目维护者，可能面临由项目领导层其他成员决定的临时或永久性后果。
